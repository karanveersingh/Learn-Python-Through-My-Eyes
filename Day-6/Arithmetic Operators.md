# Arithmetic Operators
Arithmetic operators are used to perform mathematical operations in Python. Here are the basic arithmetic operators:

| Operator | Name           | Example    |
|----------|----------------|------------|
| `+`      | Addition       | `a + b`    |
| `-`      | Subtraction    | `a - b`    |
| `*`      | Multiplication | `a * b`    |
| `/`      | Division       | `a / b`    |
| `//`     | Floor Division | `a // b`   |
| `%`      | Modulus        | `a % b`    |
| `**`     | Exponentiation | `a ** b`   |

**Example:**

```python
a = 10
b = 3

print(a + b)   # 13
print(a - b)   # 7
print(a * b)   # 30
print(a / b)   # 3.333...
print(a // b)  # 3
print(a % b)   # 1
print(a ** b)  # 1000
```
